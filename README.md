## 📋 Setup EmailJS (formulaire fonctionnel)
1. Créez compte gratuit [emailjs.com](https://www.emailjs.com)
2. Ajoutez service Email (Gmail/Outlook)
3. Créez template avec champs: from_name, from_email, subject, message
4. Éditez `js/main.js`: remplacez `YOUR_SERVICE_ID`, `YOUR_TEMPLATE_ID`, `YOUR_PUBLIC_KEY`
5. Test: `start contact.html`

**Alternative** Netlify Forms: drag dossier sur [netlify.com/drop](https://app.netlify.com/drop), form auto-func no JS.
